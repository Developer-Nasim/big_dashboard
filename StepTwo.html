Step 2. Menu Categories

The 2nd step is to create your Menu Categories. Menu Categories incluide sections like appetizers, soups, sandwiches, etc. When creating a Menu Category, make sure to select the number of sizes for that category. For example, a category like soups might come in 3 sizes: small, medium, large. If a category include items that are all in one size, then select 1 for Size.
